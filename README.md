# LoveKWalmartCode

Simple coding challenge for Walmart
Displays a list of 249 countries on a scrollable screen

Requirements:

Android Studio IDE
Minimum SDK: 24

Libraries:

Kotlin Coroutines
Recyclerview
Mockito
Retrofit
Okhttp
